**mapbox-postting**

基于 mapbox-gl 封装的一些通用地图交互功能，如撒点、热力图、轨迹播放、区域绘制等；

# 安装与使用

## 安装

使用npm进行安装

```
npm install mapbox-postting --save
```

## 使用

按需导入进行使用

```js
import 'mapbox-postting/dist/index.css' // 导入css样式
import { CirclePointLayer } from 'mapbox-postting'
```
